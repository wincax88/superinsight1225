# SuperInsight 企业级管理前端 - 实施任务计划

**当前实现状态**: 98% 完成 - 所有核心业务功能已实现，包括认证、注册、仪表盘、任务管理、账单结算、设置页面、Label Studio 集成、管理员控制台、数据增强、质量管理和安全审计。完整的服务层、类型定义和 React Query Hooks 已就绪

## 技术栈

- **前端框架**: React 18 + Vite + TypeScript
- **UI 组件库**: Ant Design Pro + Pro Components
- **状态管理**: Zustand + TanStack Query
- **路由**: React Router v6
- **国际化**: i18next + react-i18next
- **测试**: Jest + React Testing Library + Playwright

## 开发指南

### 环境要求
- Node.js 18+
- npm 或 yarn
- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

### 快速开始
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行测试
npm run test

# 运行 E2E 测试
npm run test:e2e
```

### 开发规范
- 使用 TypeScript 进行类型检查
- 遵循 ESLint 和 Prettier 代码规范
- 组件使用函数式组件和 Hooks
- 状态管理使用 Zustand
- API 调用使用 TanStack Query
- 样式使用 Ant Design 主题系统
