## 概述

SuperInsight 是一个企业级数据标注管理平台，提供完整的标注任务管理、质量控制、账单结算和安全审计功能。本手册将帮助您快速了解和使用系统的各项功能。

### 系统特点

- 🔐 **多租户架构** - 支持多组织隔离的数据管理
- 📊 **实时数据分析** - 可视化的仪表盘和报表
- 🏷️ **Label Studio 集成** - 专业的标注工具集成
- 💰 **智能计费** - 灵活的计费规则配置
- 🛡️ **安全审计** - 完整的操作日志追踪
- 🌐 **国际化支持** - 中英文双语界面

## 快速入门

### 系统要求

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **屏幕分辨率**: 推荐 1920x1080 或更高
- **网络**: 稳定的网络连接

### 首次登录

1. 打开系统 URL 进入登录页面
2. 输入管理员提供的用户名和密码
3. 点击「登录」按钮
4. 首次登录可能需要修改密码

## 用户认证

### 登录

1. 访问系统登录页面
2. 输入用户名和密码
3. 可选：勾选「记住我」保持登录状态
4. 点击「登录」按钮

### 注册（如果开放）

1. 点击登录页面的「注册」链接
2. 填写必要信息：
   - 用户名
   - 邮箱
   - 密码（需符合密码策略）
   - 确认密码
3. 点击「注册」完成

### 忘记密码

1. 点击登录页面的「忘记密码」链接
2. 输入注册邮箱
3. 检查邮箱获取重置链接
4. 按提示设置新密码

### 退出登录

点击右上角用户头像，选择「退出登录」

## 标注任务管理

### 任务列表

任务列表页面展示所有标注任务，支持：

- **搜索**: 按任务名称、项目搜索
- **筛选**: 按状态、优先级、创建时间筛选
- **排序**: 按创建时间、优先级、进度排序

### 任务状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 待开始 | 灰色 | 任务已创建，等待开始 |
| 进行中 | 蓝色 | 任务正在进行 |
| 审核中 | 橙色 | 标注完成，等待审核 |
| 已完成 | 绿色 | 任务全部完成 |
| 已取消 | 红色 | 任务已取消 |

### 创建任务

1. 点击「创建任务」按钮
2. 填写任务信息：
   - **任务名称**: 简洁描述任务内容
   - **任务描述**: 详细说明任务要求
   - **优先级**: 高/中/低
   - **截止日期**: 任务完成期限
   - **分配人员**: 选择执行人
3. 上传待标注数据
4. 配置标注模板
5. 点击「创建」完成

### 任务详情

点击任务进入详情页，可查看：

- 任务基本信息
- 进度统计
- 标注人员分配
- 质量统计
- 操作日志

### 进入标注

1. 在任务详情页点击「开始标注」
2. 系统将打开 Label Studio 标注界面
3. 按照标注指南完成标注
4. 提交标注结果

## 账单与结算

### 账单列表

账单页面展示所有计费记录：

- **筛选条件**: 日期范围、项目、状态
- **导出功能**: 支持导出 Excel 报表

### 账单详情

点击账单查看详细信息：

- 账单基本信息
- 费用明细
- 工时统计
- 标注数量

### 报表功能

#### 生成报表

1. 选择日期范围
2. 选择报表类型：
   - 概览报表
   - 详细报表

   - 项目分析

1. 生成报表后，点击「导出报表」
2. 选择导出格式（JSON/Excel）
3. 下载报表文件

### 工时排行榜

展示标注人员的工时排名：

- 本周排名
- 本月排名
- 本季度排名

点击用户可查看详细工时记录。

### 成本分析

可视化展示成本构成：

- 按项目分布的饼图
- 按部门分布的柱状图
- 成本趋势折线图

## 质量管理

### 质量规则

#### 规则列表

展示所有质量检查规则：

| 字段 | 说明 |
|------|------|
| 规则名称 | 规则的描述性名称 |
| 类型 | 文本长度、格式验证等 |
| 严重程度 | 错误/警告/信息 |
| 状态 | 启用/禁用 |

#### 创建规则

1. 点击「创建规则」
2. 填写规则信息：
   - 规则名称
   - 规则类型
   - 严重程度
   - 检查配置
3. 保存规则

#### 运行检查

- 单个规则检查：点击规则行的「运行」按钮
- 全部规则检查：点击「运行所有规则」

### 问题跟踪

#### 问题列表

展示质量检查发现的问题：

- **筛选**: 按状态、严重程度、规则筛选
- **操作**: 修复、忽略、分配

#### 问题状态

| 状态 | 说明 |
|------|------|
| 待处理 | 新发现的问题 |
| 已修复 | 问题已解决 |
| 已忽略 | 问题已忽略 |

#### 处理问题

1. 点击问题查看详情
2. 查看问题位置和原因
3. 选择操作：
   - 修复：修改标注数据
   - 忽略：标记为忽略
   - 分配：分配给特定人员

### 质量评分

系统根据质量检查结果计算质量评分：

- 0-60: 需要改进（红色）
- 60-80: 基本合格（橙色）
- 80-100: 优秀（绿色）

## 常见问题

### Q1: 无法登录怎么办？

**可能原因和解决方案：**

1. **密码错误**: 使用「忘记密码」功能重置
2. **账号被禁用**: 联系管理员启用账号
3. **IP 被封禁**: 联系管理员解除封禁
4. **浏览器问题**: 清除缓存或更换浏览器

### Q2: 标注界面加载不出来？

**解决方案：**

1. 检查网络连接
2. 清除浏览器缓存
3. 禁用浏览器扩展插件
4. 尝试使用隐私模式

### Q3: 报表导出失败？

**解决方案：**

1. 检查数据量是否过大
2. 缩小日期范围重试
3. 检查浏览器下载设置
4. 联系管理员检查服务状态

### Q4: 数据显示不正确？

**解决方案：**

1. 刷新页面
2. 检查筛选条件
3. 检查时区设置
4. 清除浏览器缓存

### Q5: 如何批量操作？

**操作步骤：**

1. 在列表页勾选多个项目
2. 点击「批量操作」按钮
3. 选择要执行的操作
4. 确认执行
